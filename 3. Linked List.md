# 3. Linked List

## 1). Singly Linked List

* #### Definition

  A linked list is a sequential list of nodes that hold data which point to other nodes also conatining data. A singly linked list hold a reference to the next node. In implementation, always maintain a reference to the head of the linked list & a reference to the tail of the linked list for quick 

* #### Terminology

  * Head: first node in a  linked list
  * Tail: last node in a linked list
  * Pointer: reference to another node
  * Node: object containing data & pointer
